# TechFusion Enterprise Network Design

## Overview

This repository contains the planning and design documentation for the TechFusion Enterprise Network project.

The project aims to design a scalable enterprise network for a fictional multi-site organisation, including a headquarters location and a branch office. The network follows a hierarchical design model consisting of core, distribution, and access layers.

---

## Current Project Status

Project Phase: Planning and Design

Completed:

* Logical network topology design
* VLAN design
* Device inventory
* Port mapping documentation

## Network Overview

### Headquarters

The headquarters network consists of:

* Edge Router
* Multilayer Switch
* Two Core Switches
* Department Access Switches
* Server Farm

### Branch Office

The branch office consists of:

* Branch Router
* Branch Access Switch
* End-user Devices
